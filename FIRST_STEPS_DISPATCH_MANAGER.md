# First Steps for Dispatch Manager
**Role:** Dispatch & Fulfillment Manager  
**Access Level:** Orders, dispatch, courier management, address verification

---

## Overview
As a Dispatch Manager, you're responsible for ensuring orders are picked, packed, and shipped efficiently and accurately. This guide covers your first 21 days.

   - Review current dispatch queue

3. **Verify Navigation Access**
   You should have access to:
   - ‚úÖ Dashboard (dispatch metrics)
   - ‚úÖ Orders (all order statuses)
   - ‚úÖ Dispatch Dashboard (main workspace)
   - ‚úÖ Address Verification (flagged addresses)
   - ‚úÖ Courier Management (booking, tracking)
   - ‚úÖ Fraud Reporting (review flagged orders)

4. **Update Personal Settings**
   - Configure notification preferences
   - Set up mobile alerts for urgent orders
   - Update contact information

**‚úÖ Milestone:** Access verified and ready to work

## Phase 2: Order Pipeline Understanding (Days 1-3)

### Day 1-2: Order Status Workflow
**Estimated Time:** 2-3 hours

1. **Navigate to Orders ‚Üí Order Dashboard**
   - Understand order statuses and your role:
     
   **Not Your Responsibility (Yet):**
   - **Pending:** New orders, waiting for fraud check
   - **On Hold:** Flagged for fraud, Super Manager review
   
   **Your Responsibility Starts Here:**
   - **Address Clear:** Verified, ready for you to pick
   - **Ready to Dispatch:** You've picked/packed, ready for courier
   - **Dispatched:** You've handed to courier
   
   **After Your Role:**
   - **In Transit:** Courier delivering
   - **Delivered:** Customer received
   - **Cancelled:** Order cancelled

2. **Understand Stock Reservation**
   - ‚úÖ **AUTOMATIC:** Stock reserved when order is created
   - Reserved stock is NOT available for new orders
   - Stock is only deducted when YOU dispatch the order
   - This is critical: Dispatch = Inventory deduction

3. **Review Current Pipeline**
   - Count orders in each status
   - Note oldest orders (prioritize these)
   - Check for urgent/priority orders
   - Identify bottlenecks

### Day 2-3: Order Details and Prioritization
**Estimated Time:** 2-3 hours

1. **Review Individual Orders**
   - Click on any "Address Clear" order
   - Review order details:
     - Customer name and contact
     - Delivery address
     - Products ordered with quantities
     - Special instructions
     - Payment status (COD vs Paid)
     - Order value

2. **Learn Prioritization Rules**
   Priority order (high to low):
   1. **Express orders:** Customer paid for expedited shipping
   2. **High-value orders:** Valuable customers, large amounts
   3. **COD orders:** Cash on Delivery (time-sensitive)
   4. **Old orders:** Orders waiting longest
   5. **Standard orders:** Regular orders

3. **Check Fraud Flags**
   - Some orders may have fraud warnings
   - Red flag: Don't dispatch until Super Manager approves
   - Yellow flag: Extra verification during picking
   - Green: No issues, proceed normally

**‚úÖ Milestone:** Understanding of order pipeline

## Phase 3: Dispatch Workflow Mastery (Days 3-7)

### Day 3-4: Picking Process
**Estimated Time:** 3-4 hours of practice

1. **Navigate to Dispatch ‚Üí Ready to Pick**
   - View orders ready for picking
   - Select highest priority orders first
   - Print picking list or use mobile device

2. **Perform Picking**
   - Go to product locations
   - Scan barcode to verify correct item
     - ‚úÖ Correct beep: Right product
     - ‚ùå Error beep: Wrong product, rescan
   - Check quantity carefully
   - Verify product condition:
     - No damage
     - Correct size/color/variant
     - All accessories included
   - Place in picking bin/cart
   - Mark item as picked in system

3. **Quality Checks During Picking**
   - **Correct Product:** Matches order exactly
   - **Correct Quantity:** Count carefully
   - **Good Condition:** No defects or damage
   - **Complete:** All accessories/parts included

4. **Handle Picking Issues**
   
   **Out of Stock:**
   - Check if stock is reserved elsewhere
   - Check other locations (if multi-outlet)
   - Contact Super Manager immediately
   - System should prevent this, but verify
   
   **Damaged Product:**
   - Do not pick damaged item
   - Document damage with photo
   - Find replacement
   - Report damage to inventory team
   
   **Wrong Product in Location:**
   - Verify by SKU/barcode
   - Find correct product
   - Report location error

### Day 5-6: Packing Process
**Estimated Time:** 3-4 hours

1. **Set Up Packing Station**
   - Packaging materials:
     - Boxes (various sizes)
     - Bubble wrap
     - Packing tape
     - Packing peanuts/paper
     - Labels and printer
   - Tools:
     - Barcode scanner
     - Scale (for weight)
     - Tape dispenser
     - Markers

2. **Pack Order**
   - Select appropriate box size
   - Protect fragile items with bubble wrap
   - Arrange items efficiently
   - Fill empty space (prevent shifting)
   - Include:
     - Invoice/packing slip
     - Return instructions (if applicable)
     - Promotional materials (if any)
   - Seal box securely
   - Weigh package

3. **Quality Check Before Sealing**
   - ‚úÖ All items from picking list included
   - ‚úÖ Items protected adequately
   - ‚úÖ Invoice inside
   - ‚úÖ No damaged items
   - ‚úÖ Correct quantity of each item

4. **Label Package**
   - Print shipping label
   - Attach to package
   - Verify:
     - Customer name
     - Delivery address
     - Phone number
     - Order number
   - Mark package as "Ready to Dispatch"

### Day 7: Dispatch in System
**Estimated Time:** 2-3 hours

1. **Navigate to Dispatch ‚Üí Ready to Dispatch**
   - View packed orders ready for courier
   - Verify all picked and packed

2. **Mark Order as Dispatched**
   - Open order
   - Click "Mark Dispatched"
   - Enter required information:
     - Dispatch date/time
     - Package weight
     - Box dimensions (if needed)
     - Number of packages (if multiple)
   - Click "Confirm Dispatch"
   - ‚úÖ **AUTOMATIC:** Inventory deducted immediately
   - Customer receives dispatch notification

3. **Understand What Happens**
   - Stock is deducted from outlet inventory
   - Order status changes to "Dispatched"
   - Customer gets tracking notification
   - Order appears in courier handover list

**‚úÖ Milestone:** Complete dispatch workflow mastered

## Phase 4: Courier Integration (Days 7-10)

### Day 7-8: Understanding Courier System
**Estimated Time:** 2-3 hours

1. **Navigate to Dispatch ‚Üí Courier Management**
   - Review configured couriers:
     - Postex
     - Leopard Courier
     - TCS
     - Others
   - Understand courier capabilities:
     - Coverage areas
     - Service levels (standard, express)
     - Weight limits
     - Pricing

2. **Learn Rate Comparison**
   - System shows rates from multiple couriers
   - Compare by:
     - Price
     - Delivery time
     - Service quality
     - Coverage of destination
   - Select best option for each order

### Day 8-9: Book Courier Pickups
**Estimated Time:** 2-3 hours

1. **Navigate to Dispatch ‚Üí Courier Booking**
   - Select dispatched orders
   - Click "Book Courier"
   - System shows rate comparison

2. **Create Booking**
   - Select courier
   - Verify package details:
     - Weight
     - Dimensions
     - Destination
     - Declared value
   - Choose service level:
     - Standard (cheaper, slower)
     - Express (expensive, faster)
   - Set pickup date/time
   - Add special instructions if needed
   - Click "Confirm Booking"

3. **Generate Tracking Number**
   - System generates courier tracking number
   - Attach to package
   - Update order with tracking info
   - Customer receives tracking notification

4. **Prepare for Courier Pickup**
   - Group packages by courier
   - Print manifest/handover sheet
   - Count packages
   - Stage near pickup area
   - Be ready at scheduled time

### Day 9-10: Courier Handover
**Estimated Time:** Varies

1. **When Courier Arrives**
   - Verify courier identity
   - Provide manifest
   - Hand over packages
   - Courier scans packages
   - Get receipt/signature
   - Keep copy for records

2. **Update System**
   - Mark orders as "Handed to Courier"
   - Update pickup time
   - Upload receipt/photo (if required)
   - Any issues documented

3. **Handle Handover Issues**
   
   **Courier Rejects Package:**
   - Common reasons:
     - Overweight
     - Improper packaging
     - Restricted items
     - Wrong destination
   - Resolve issue:
     - Repack if needed
     - Choose different courier
     - Contact customer if undeliverable
   
   **Missing Packages:**
   - Verify against manifest
   - Check staging area
   - Update manifest
   - Document discrepancy

**‚úÖ Milestone:** Courier integration mastered

## Phase 5: Address Verification (Days 10-14)

### Day 10-11: Understanding Address Flags
**Estimated Time:** 2-3 hours

1. **Navigate to Address Verification**
   - Review flagged addresses
   - Understand flag reasons:
     - üö© Incomplete address (missing details)
     - üö© Unrecognized location (system can't validate)
     - üö© Outside service area (no courier coverage)
     - üö© Ambiguous landmarks (unclear directions)
     - üö© Conflicting information (name vs address)

2. **Review Flagged Orders**
   - Check each flagged order
   - Read address details
   - Review customer contact information
   - Note specific issues

### Day 11-13: Address Verification Process
**Estimated Time:** 2-3 hours

1. **Contact Customer**
   - Call customer (preferred method)
   - Use WhatsApp if configured
   - SMS as backup
   - Email as last resort (slow response)

2. **Verify Address Details**
   Ask customer to confirm:
   - House/building number
   - Street name
   - Area/neighborhood
   - Nearest landmark
   - City and postal code
   - Alternative contact number
   - Best time for delivery

3. **Update Address in System**
   - Navigate to order
   - Click "Edit Address"
   - Update with verified details
   - Add landmark in notes
   - Mark "Address Verified"
   - Click "Mark Address Clear"
   - Order moves to dispatch queue

4. **Handle Uncontactable Customers**
   - Try multiple contact methods
   - Try multiple times (different hours)
   - Wait 24-48 hours
   - If still no response:
     - Put order "On Hold"
     - Escalate to Super Manager
     - May need to cancel order

### Day 13-14: Prevent Future Address Issues

## Critical Understanding Points

### ‚úÖ Automatic Inventory Update
**Your dispatch action deducts inventory:**

- **When you mark order "Dispatched":** ‚úÖ Stock deducted immediately
- This is THE critical moment for inventory
- If you dispatch wrong item, inventory will be wrong
- Always verify items before dispatching

### üìä Key Metrics You Own
- **Dispatch Speed:** Average time from "Address Clear" to "Dispatched"
- **Dispatch Accuracy:** % of orders with correct items
- **Damage Rate:** % of orders damaged in shipping
- **Courier Performance:** On-time delivery rates
- **Cost per Dispatch:** Average shipping cost per order

### üö® Escalation Criteria
**When to escalate to Super Manager:**
- Cannot fulfill order (out of stock)
- High-value order damage
- Courier consistently poor performance
- Address verification unsuccessful after 48 hours
- Customer complaints about dispatch
- System technical issues preventing dispatch

## Pro Tips for Success

### Picking Accuracy
- **Always scan:** Never pick by visual alone
- **Double-check quantities:** Count carefully
- **Verify variants:** Size, color, version matter
- **Check condition:** Don't ship damaged items

### Packing Efficiency
- **Right box size:** Not too big, not too small
- **Protect well:** Customer's first impression is package
- **Pack tight:** Prevent shifting during transit
- **Label clearly:** Wrong address = delayed delivery

### Courier Management
- **Build relationships:** Good rapport gets better service
- **Track performance:** Data drives decisions
- **Communicate issues:** Don't let problems fester
- **Be flexible:** Have backup couriers

### Address Verification
- **Call first:** Fastest method
- **Be clear:** Explain why you're calling
- **Note landmarks:** GPS isn't always accurate
- **Verify spelling:** Avoid wrong deliveries

## Common Challenges & Solutions

### Challenge: Picking wrong items
**Solution:**
- Always use barcode scanner
- Verify SKU matches order
- Implement double-check system
- Review high-value orders twice

### Challenge: Orders stuck in "Address Clear"
**Solution:**
- Set daily dispatch targets
- Prioritize old orders
- Batch pick similar orders
- Add temporary staff if needed

### Challenge: Courier delays pickups
**Solution:**
- Schedule pickups earlier in day
- Have backup courier ready
- Drop off at courier hub if urgent
- Escalate to courier account manager

### Challenge: High return rate due to wrong items
**Solution:**
- Implement quality check before dispatch
- Take photos of packed items
- Train staff on accuracy importance
- Review and learn from each error
