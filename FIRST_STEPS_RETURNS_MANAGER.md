## Phase 5: Refund Processing (Days 14-18)

### Day 14-15: Calculate Refunds
**Estimated Time:** 2-3 hours

1. **Navigate to Returns â†’ Pending Refunds**
   - View returns awaiting refund
   - System shows calculated refund

2. **Verify Refund Calculation**
   
   **Full Refund Calculation:**
   - Original product price
   - \- Any discounts applied at purchase
   - + Shipping charges (if eligible)
   - = Total refund
   
   **Partial Refund Calculation:**
   - Original product price
   - Ã— Condition % (e.g., 70% for minor damage)
   - = Refund amount
   
   **Deductions:**
   - Restocking fee (if policy)
   - Missing accessories value
   - Repair cost for damage
   - Return shipping (if policy)

3. **Review Special Cases**
   
   **Promotional Items:**
   - If order had "Buy 1 Get 1 Free"
   - Customer returns the free item
   - No refund (item was free)
   
   **Bundled Products:**
   - Customer returns part of bundle
   - Recalculate at non-bundle price
   - May result in lower refund
   
   **Gift Cards Used:**
   - Refund back to gift card
   - Not to original payment method

### Day 15-17: Initiate Refunds
**Estimated Time:** 2-3 hours

1. **Process Refund**
   - Open return record
   - Click "Process Refund"
   - Verify refund amount
   - Select refund method:
     - **Original Payment Method:** Credit/debit card
     - **Cash:** For COD orders
     - **Store Credit:** Customer keeps credit for future
     - **Bank Transfer:** For large amounts
   - Enter refund reference number
   - Add notes
   - Click "Confirm Refund"

2. **Refund Timelines**
   - **Cash:** Immediate (in person)
   - **Store Credit:** Immediate
   - **Credit Card:** 5-10 business days
   - **Bank Transfer:** 2-5 business days
   - Customer receives notification with timeline

3. **Handle Refund Issues**
   
   **Customer Hasn't Received Refund:**
   - Check processing status
   - Verify bank details
   - Contact payment processor
   - Provide refund reference number
   - Escalate if needed
   
   **Wrong Refund Amount:**
   - Review calculation
   - Check if additional deductions
   - Explain breakdown to customer
   - Adjust if error found
   
   **Customer Wants Different Method:**
   - Original method is standard
   - Store credit as alternative
   - Escalate to Super Manager for exceptions

## Common Challenges & Solutions

### Challenge: Customer disputes condition assessment
**Solution:**
- Share clear photos
- Explain policy objectively
- Offer compromise if reasonable
- Escalate to Super Manager if needed

### Challenge: Delayed returns (customer approved but doesn't ship)
**Solution:**
- Reminder after 3 days
- Second reminder after 7 days
- Cancellation notice after 14 days
- Close return request after 21 days

### Challenge: Return different item than ordered
**Solution:**
- Document with photos
- Contact customer for correct item
- Hold refund until resolved
- File fraud report if intentional

### Challenge: High return rate for specific product
**Solution:**
- Analyze return reasons
- Identify root cause (quality, sizing, description)
- Report to Super Manager
- Recommend action (improve listing, contact supplier, discontinue)
